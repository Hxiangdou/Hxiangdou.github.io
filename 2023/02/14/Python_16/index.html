JSON文件

JSON(JavaScript Object Notation) 是一种轻量级的数据交换格式。JSON在python中分别由list和dict组成。JSON模块提供了四个功能：dumps、dump、loads、load。

dumps

dumps将python中的字典转换为字符串。

loads

loads将字符串转换为字典。

dump

dump将数据写入json文件中。

load

load把文件打开，并把字符串变换为数据类型。
